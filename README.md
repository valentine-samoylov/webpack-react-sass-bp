# Webpack Boilerplate (React + SASS Edition)

## Features

- Uses Yarn
- Webpack
- Babel transpiller
- React
- SASS(SCSS) preprocessor
- PostCSS
- Images optimization

## Scripts

To start development mode:

```sh
yarn run start
```

To a create production build:

```sh
yarn run build
```

To lint code:

```sh
yarn run lint
```

To prettify code:

```sh
yarn run prettify
```
